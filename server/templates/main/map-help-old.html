{{ _('How to draw your search area') }}

{{ _('Click the add area button') }}

  • {{ _('choose a location on the map and click to start drawing') }}
  • {{ _('keep clicking around the edge of your search area to draw the extent') }}
  • {{ _('to complete your shape, join the ends up or double click') }}

{{ _('To draw curves') }}

  • {{ _('hold down the SHIFT key and click') }}
  • {{ _('release the SHIFT key and mouse button to stop drawing') }}
  • {{ _('or release only SHIFT to continue drawing a straight edge') }}
