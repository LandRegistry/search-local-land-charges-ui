{{ _('How to draw your search area') }}

  1. {{ _('Select the draw area radio button.') }}
  2. {{ _('Select a point on the map to start drawing.') }}
  3. {{ _('Keep selecting points to draw your search area.') }}
  4. {{ _('Select the first point again, or double click anywhere, to complete your search area.') }}

{{ _('To draw curves') }}

  1. {{ _('Hold down the SHIFT key and select.') }}
  2. {{ _('Release the SHIFT key and mouse button to stop drawing, or release only SHIFT to continue drawing a straight edge.') }}
